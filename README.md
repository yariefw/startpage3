# Startpage

## Features

- Organized bookmarks with search
- Custom wallpaper with adjustable opacity
- Day, hour, and work time progress bars
- Encrypted bookmarks support
- Settings can be saved and auto loaded from local storage


## Disclaimer

AI tools were used for testing purposes only during the development of this project. Every bit of code and every stupid mistake are of my own making.
